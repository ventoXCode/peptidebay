# PeptideBay 🧬💊

**Everything you need to know about peptides and supplements — all in one place.**

PeptideBay is a comprehensive, searchable reference guide covering peptides and supplements with details on benefits, dosages, side effects, and research status.

## Features

- **47 entries** covering popular peptides and supplements
- **Search** by name, benefit, tag, or keyword
- **Filter** by type (peptides / supplements) and subcategory
- **Detailed modal** for each entry including dosage, benefits, side effects, and research status
- **Fully static** — no build tools or dependencies required; open `index.html` in any browser
